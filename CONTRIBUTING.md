# Contributing to the Marketing Automation Dashboard

We love seeing contributions from the community!

## Getting Started

- Fork the repository and clone it locally.
- Create a new branch for your contribution.
- Make your changes and commit them with a descriptive commit message.
- Push your changes to your fork and open a pull request.

## Code Style

- We use the [PEP 8](https://www.python.org/dev/peps/pep-0008/) style guide for Python code.
- Please make sure your code is well-documented and includes tests.

## Issues

- If you find a bug, please open an issue and provide as much detail as possible.
- If you have a feature request, please open an issue and describe the feature in detail.
